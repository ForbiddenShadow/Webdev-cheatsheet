CSS cheatsheet

Common selectors

Selector Description
ID The most specific form of selector, IDs can be defined once in an HTML document
class A level above in specificity, classes can be defined and used as much as you want
Element Selector This will select all elements of the same type, is the least specific selector

Box Model

Property Description
Width How much space the content takes vertically
Height How much space content takes up horizontally
Margin How much room there is between content
Padding Sits around the corner of the content like whitespace
